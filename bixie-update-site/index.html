Eclipse Update Site

This is the Eclipse Update Site for Bixie, the inconsistent code detection tool for Java.

Plugin Requirements

The Bixie plugin requires at least Java 1.7. The plugin has only been tested with the Eclipse versions Kepler and Luna. Older version may work but are not actively supported.

Installation

  • In Eclipse, click on Help -> Install New Software.
  • In "Work with" paste the URL http://www.csl.sri.com/users/schaef/update-site/ and hit Enter.
  • Select Bixie in the list below and hit Next.
  • Follow the instructions and complete the installation.
If the installation was successful, you should see a Bixie Icon in your toolbar. Click the button to toggle Bixie warnings in on and off in the current Java file.
